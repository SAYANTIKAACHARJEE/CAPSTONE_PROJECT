# 🛒 E-Commerce Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive Power BI dashboard that analyzes e-commerce sales data. It helps in understanding sales trends, customer behavior, and overall business performance through visualizations.

---

## 🎯 Problem Statement
In the e-commerce industry, large amounts of sales data are generated daily. It becomes difficult to analyze this data manually and extract meaningful insights for business decision-making.

---

## 💡 Solution
This project provides a Power BI dashboard that visualizes sales data using charts and graphs. It helps in identifying trends, top-performing products, and regional sales performance.

---

## 🛠️ Tools & Technologies Used
- Power BI  
- Power Query  
- Data Visualization Techniques  

---

## 📊 Features of Dashboard
- Total Sales and Profit Analysis  
- Region-wise Sales Performance  
- Category-wise Analysis  
- Top-Selling Products  
- Interactive Filters and Slicers
